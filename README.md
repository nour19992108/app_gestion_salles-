## Description
Ce projet consiste à concevoir et développer une application Python de gestion des salles, avec une interface graphique (GUI) et une base de données MySQL.

L’application suit une architecture en couches afin d’assurer une séparation claire entre l’accès aux données, la logique métier et l’interface utilisateur.
